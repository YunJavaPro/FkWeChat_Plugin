# 涛神自制

## 插件简介

涛神自制用于隐藏/移除 FkWeChat 在微信聊天页面自动创建的“快捷工具”悬浮窗。

插件会在运行时调用 FkWeChat 模块自身的悬浮窗移除逻辑，并配合分阶段轮询策略持续压制页面切换后重新出现的快捷工具悬浮窗。

## 主要功能

- 自动隐藏 FkWeChat 快捷工具悬浮窗
- 页面返回、重新进入聊天后继续生效
- 写入悬浮窗禁用相关配置，减少自动展开
- 分阶段轮询，降低长期后台消耗
- 限频日志，避免异常时频繁刷屏

## 轮询策略

- 启动后 0-5 秒：80ms 高频压制，减少刚进入页面时的闪现
- 启动后 5-30 秒：300ms 中频维持
- 启动 30 秒后：1000ms 低频保活，降低长期消耗

## 适用场景

适用于不需要 FkWeChat 快捷工具悬浮窗，并希望聊天页面保持干净的用户。
