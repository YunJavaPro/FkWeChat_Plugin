# 关键词监听

监听指定联系人或群聊中的关键词，匹配后发送提醒通知。

## 功能

- 监听指定好友、群聊，或监听全部消息
- 支持多个关键词，使用 `|` 分隔，匹配时不区分大小写
- 支持弹窗、日志、管理员私聊、自己接收、指定群聊、原群 @ 发送者等提醒方式
- 支持通过聊天命令管理监听目标、关键词、提醒模式和提醒群

## 常用命令

```text
#帮助                 查看帮助
#开启 / #关闭          开启或关闭监听
#状态                 查看当前配置
#监听 [名称/wxid]      添加监听目标
#监听 全部             监听全部消息
#取消监听 [名称/wxid]  移除监听目标
#取消监听 全部         取消监听全部消息
#监听列表              查看监听目标
#关键词 [词]           添加关键词
#关键词 a|b|c          批量添加关键词
#删词 [词]             删除关键词
#词列表                查看关键词
#模式 [模式名]         设置提醒模式
#设管理员              在私聊中设置管理员
#设提醒群              在群聊中设置提醒群
#查找 [名称]           搜索联系人或群聊
#清空                 重置配置
```

## 提醒模式

```text
toast  弹窗提示并记录日志
log    仅记录日志
admin  私聊发送给管理员
self   发送给自己
group  发送到指定提醒群
at     在原群聊中 @ 发送者
all    同时使用以上方式
```

## 配置文件

`config.prop` 提供默认配置模板。安装后也可以通过聊天命令修改配置。
