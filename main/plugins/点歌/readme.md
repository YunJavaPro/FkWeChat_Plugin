# 🎵 点歌插件

基于 FkWeChat 框架的多音源点歌插件
默认flac再高不兼容，顺延能搜到的最高音质。
网易云是mp3，源4有歌词所以默认源4。

## ✨ 功能特性
- **4个网站**：
- https://api.mmp.cc 远梦
- http://api.dovis.work 云烟
- https://api.nki.pw 西瓜糖
- https://api.suol.cc 慕名

- **多选音源**：同时搜索多个音源，自动合并结果
- **音乐卡片**：发送带封面的音乐卡片
- **直链文本**：发送歌曲直链+详细信息
- **歌单列表**：搜索结果列表，回复数字选歌
- **命令授权**：群聊中可指定成员使用管理命令
- **自定义卡片**：自定义歌手名、封面图
- **首尾文案**：自定义直链文本的前缀后缀
- **搜索发送**：UI内直接搜索并发送

## ⚠️ 注意

- MV音源只发直链文本，不发卡片
- 自定义歌手/封面会覆盖原信息
- 群聊命令授权需手动勾选成员

## 📋 命令
1. 长按任意消息 → 打开设置界面
2. 选择音源、配置开关
3. 群聊中可授权指定成员使用管理命令
- | 命令 | 说明 |
- |------|------|
- | `切换 数字` | 切换音源（1-8，0=全选） |
- | `切换` | 显示帮助 |
- | `音乐开关` | 开关点歌功能 |
- | `群聊开关` | 开关群聊响应 |
- | `卡片开关` | 开关音乐卡片 |
- | `直链开关` | 开关直链文本 |
- | `列表开关` | 开关歌单列表 |
- | `日志开关` | 开关调试日志 |
- | `指令 触发词` | 修改点歌触发词 |
- | `句首 文本` | 设置直链前缀 |
- | `句尾 文本` | 设置直链后缀 |
- | `列表` | 显示歌单列表 |
- | `数字` | 选择列表中的歌曲 |
- | `点歌 歌名` | 点歌（触发词可自定义）
